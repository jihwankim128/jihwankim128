### 만약이라는 키워드로 다양한 시나리오에 대응하는 백엔드 개발자 If입니다.

* 만약, 내가 협업하는 관계라면 어떤 사람과 함께하고 싶을까?  
* 만약, 현재 기술에 대해 부채가 발생하진 않을까? 이렇게 리팩토링 하면 어떨까?
